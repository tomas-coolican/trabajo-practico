# Proyecto: Sistema de Alquiler de Vehículos

## Objetivo
Desarrollar un sitio web estático usando únicamente HTML5 y CSS3.

## Restricciones
- NO JavaScript
- NO Bootstrap
- NO Tailwind
- NO librerías externas

## Páginas requeridas

### 1. Landing Page
Debe incluir:
- Header
- Hero section
- Servicios
- Vehículos destacados
- Sobre nosotros
- Footer

### 2. Reserva
Debe incluir:
- Formulario completo
- Datos personales
- Datos de reserva
- Opciones adicionales
- Comentarios

## Requisitos técnicos
- Responsive hasta 300px
- Flexbox o CSS Grid
- Etiquetas semánticas
- Media queries

## Entrega
- Repositorio GitHub
- Deploy funcional
